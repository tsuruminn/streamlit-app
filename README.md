消費者物価指数
2020年基準消費者物価指数 / 長期時系列データ / 品目別価格指数 / 東京都区部 / 年平均

データセット名または表番号：

各列の意味：
EXCELで使わない行の削除を行った。
一列目に年、それ以降の列は各品目名が連なっていて、年ごとの消費者物価指数が入っている。

単位・注意点：
折れ線グラフ
横軸：年
縦軸：なし

相関分析は品目を二つ選択しているときしかできないことに注意してください。

未使用UI部品の利用
●st.toggle
チュートリアルを用意したときに長くなったので、理解し終わったら非表示にできるようにした方が使いやすくなると思ったため利用した。
アプリのタイトルの真下にトグルをオンにした状態で配置している。

●st.spinner
グラフの読み込みが遅くなった時にしっかりと読み込みが始まっているかどうかを確認でき、安心させる目的で利用した。
グラフを読み込む場面で使っている。
●st.info
警告文をst.writeやst.subheaderを使って表示すると利用者の読み逃しが起きそうだったので使った。
本アプリの相関分析は、品目を二種類選択したときしか表示されないため、品目をそれ以上やそれ以下しか選択していない場合に表示されるようにif文で使われている。
